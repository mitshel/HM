## Что такое Home Media catalog
HMc - программа для каталогизации домашний архивов медиа-файлов. В первую очередеь - фотоальбомы и семейные видеоролики.
Разработана для использования на NAS и небольших домашних Linux - серверов. Основным ее отличием от аналогичных ктаалогизаторов
является каталогизация путем сканирования структуры каталогов файлового архива, посредством которой уже создана схема хранения и
именования архивов.
